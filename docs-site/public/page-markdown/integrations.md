# Integrations

This section covers the supported ways ReCursor connects to surrounding tooling.

- [Claude Code Hooks](/integrations/claude-code-hooks/) for one-way event observation.
- [Agent SDK](/integrations/agent-sdk/) for parallel sessions that ReCursor can drive.
- [OpenCode UI patterns](/integrations/opencode-ui-patterns/) for desktop-to-mobile UX parity references.

> ReCursor does **not** claim support for joining or mirroring first-party Claude Code Remote Control sessions.
