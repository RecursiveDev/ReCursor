Core Concepts

Data Flow

Message-level flows between the mobile app, bridge server, and Claude Code.

User Sends a Message

  ReCursor App       Bridge Server       Agent SDK         Claude API
      |                    |                 |                  |
      |--- message ------->|                 |                  |
      |                    |-- agent.run() ->|                  |
      |                    |                 |--- API request ->|
      |                    |                 |<-- stream -------|
      |                    |<- stream_chunk -|                  |
      |<-- stream_chunk ---|                 |                  |
      |    (update UI)     |                 |                  |
      |                    |<- complete -----|                  |
      |<-- stream_end -----|                 |                  |

Tool Execution with Approval

  ReCursor App       Bridge Server       Agent SDK         Tools
      |                    |                 |               |
      |                    |                 |-- tool_use -->|
      |                    |<- tool_use -----|               |
      |<-- approval_req ---|                 |               |
      |                    |                 |               |
      |-- approved ------->|                 |               |
      |                    |-- continue ---->|               |
      |                    |                 |-- execute --->|
      |                    |                 |<-- result ----|
      |                    |<- tool_result --|               |
      |<-- tool_result ----|                 |               |
      |  (render card)     |                 |               |

Hook Event Observation

  ReCursor App       Bridge Server        Hooks          Claude Code
      |                    |                 |               |
      |                    |                 |<- PostToolUse-|
      |                    |<- HTTP POST ----|               |
      |                    |--- 200 OK ----->|               |
      |<-- claude_event ---|                 |               |
      |  (update timeline) |                 |               |
